---

## ユーザーペルソナ

### ペルソナ1: 贈り手（企業マーケター）

| 項目 | 内容 |
|------|------|
| 名前 | 田中さん（マーケティング担当） |
| 属性 | 30代、飲食チェーンのマーケティング部門 |
| 目標 | SNSキャンペーンで新規顧客を獲得したい |
| 課題 | 当選者への景品発送が手間、住所を聞くとCVRが下がる |

### ペルソナ2: 受け取り手（エンドユーザー）

| 項目 | 内容 |
|------|------|
| 名前 | 佐藤さん（一般消費者） |
| 属性 | 20代、スマホネイティブ世代 |
| 目標 | キャンペーンで当たったギフトを簡単に使いたい |
| 課題 | 個人情報入力は面倒、紙のチケットは無くしがち |


## MVPスコープ

### MVP機能一覧

1. **ギフト商品一覧・詳細表示**: 贈り手が商品を選択
2. **ギフトURL発行**: リアルタイムでユニークURL生成
3. **LINEシェア（liff.shareTargetPicker）**: ワンタップでLINE友だちにシェア
4. **ギフト受け取り画面**: 受け取り手がギフト内容を確認・受け取り
5. **デジタルチケット表示**: QRコード付きチケット
6. **利用完了画面**: 店舗利用後の完了表示

### 後回し機能

1. **受け取り追跡ダッシュボード**: 管理者向け分析機能
2. **ギフト履歴一覧**: 過去に受け取ったギフトの一覧
3. **有効期限アラート**: チケット期限前の通知
4. **複数商品ギフト**: 複数商品をまとめて贈る機能
