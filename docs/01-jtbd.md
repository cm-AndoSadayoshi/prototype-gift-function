# JTBD (Jobs To Be Done)

## プロダクト名: ソーシャルギフトアセット

---

## メインジョブ

### 贈り手のメインジョブ

When **友人や家族に感謝や祝福を伝えたいとき**
I want to **手軽にデジタルギフトを贈りたい**
So that **相手の住所を知らなくても、すぐに気持ちを届けられる**

### 受け取り手のメインジョブ

When **LINEでギフトURLを受け取ったとき**
I want to **簡単にギフトを受け取って使いたい**
So that **面倒な手続きなしで特典を利用できる**

---

## 関連ジョブ

### ジョブ1: 即座にギフトを届けたい（贈り手）

When 誕生日や記念日を思い出したとき
I want to その場ですぐにギフトを手配したい
So that タイミングを逃さず気持ちを伝えられる

### ジョブ2: 安全にギフトを受け取りたい（受け取り手）

When 知人からギフトURLが届いたとき
I want to 個人情報を入力せずにギフトを受け取りたい
So that 安心してギフトを利用できる

### ジョブ3: ギフトの状況を把握したい（贈り手）

When ギフトを送った後
I want to 相手が受け取ったか確認したい
So that ちゃんと届いたか安心できる

### ジョブ4: いつでも使えるようにしたい（受け取り手）

When 受け取ったギフトを使いたいとき
I want to スマホでチケットを表示したい
So that 財布やカードを持ち歩かなくても利用できる

---

## ジョブの優先順位

| 優先度 | ジョブ | 理由 |
|--------|--------|------|
| 1 | 手軽にデジタルギフトを贈りたい | コアバリュー：住所不要の即時配送 |
| 2 | 簡単にギフトを受け取って使いたい | 受け取り体験がリピートに直結 |
| 3 | ギフトの状況を把握したい | 贈り手の安心感、追客への布石 |
| 4 | 安全にギフトを受け取りたい | 信頼性の担保 |
